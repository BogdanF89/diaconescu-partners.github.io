Section C — Premium Foundries (paid web licenses) Cannot be live-previewed here — requires purchase

If the client wants a truly unique typeface no competitor is using

These are the typefaces used by the world's top law firms, consultancies and luxury brands. A web licence typically costs €150–600 one-time and guarantees no other Romanian law firm will use the same font.

  • GT Sectra — Grilli Type · grilli.ch
    Editorial serif with a sharp scalpel-cut feel. Used by The Guardian, Financial Times.
  • Tiempos Display — Klim Type · klim.co.nz
    New Zealand foundry. Refined, contemporary serif. Used by Stripe, GitHub.
  • Canela — Commercial Type · commercialtype.com
    Soft high-contrast serif. Widely used in luxury editorial.
  • Neue Haas Grotesk — Linotype · linotype.com
    The original Helvetica predecessor. The most authoritative sans ever made.
  • Portrait — Commercial Type
    Sharp bracketed serif. Stately, classical but completely contemporary.
  • Graphik — Commercial Type
    The "designer's Helvetica". Used by McKinsey, BCG, Goldman Sachs.
  • Freight Display — GarageFonts · fonts.adobe.com
    Available on Adobe Fonts (CC subscription). Elegant, high-contrast display serif.
  • Acumin Pro — Robert Slimbach · fonts.adobe.com
    Adobe Fonts. Clean, versatile grotesque with exceptional screen legibility.
